
results folder contains raw benchmarking results and code used to create figures from this data
Amdahl's law and moores_law folders are for specific figures

fft_main.py contains the main fft code
dft_figs.py create the figures for DFT input and output plots
misc.py contains some miscellaneous functions for benchmarking and checking correctness of code
